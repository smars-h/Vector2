# vector2 # 
vector2 is vector calculator

# Usage / Installation #
You can install vector2 using npm 
```
npm i @smarsh1/vector2
```

# Example #
```js
const Vector = require('@smarsh1/vector2');

let vec = new Vector(2, 3);

console.log(vec.add(5, 6));

// expected output: 7, 9 
```

# All Methods #
Add
```js
add(x, y);
```

Subtract
```js
sub(x, y);
```

Multiplcate
```js
multi(x, y)
```

Divade
```js
div(x, y)
```

Lenght
```js
len()
```

Normalize
```js
norm()
```
